The Marigold — Instagram Template Kit

Ready-to-use templates at exact Instagram dimensions. Screenshot each template or open in browser at full size and save as image. Swap the placeholder text for each post.

How to use these:
1. Right-click any template → "Open image in new tab" or screenshot it
2. For full resolution: inspect element → find the .inner div → change transform to scale(1) → screenshot at 1080px width
3. Or, zoom your browser to 400% and screenshot the template area
4. Swap the [BRACKETED TEXT] with your actual content each time
5. Keep the visual style consistent — same fonts, colors, tape/pin elements
